# 🧮 Simple Calculator

A beginner Python project — Simple Calculator with basic operations.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division

## How to Run

```bash
python calculator.py
```

## Example

```
Select operation:
1. Addition      (+)
2. Subtraction   (-)
3. Multiplication (*)
4. Division      (/)

Enter choice: 1
Enter first number: 10
Enter second number: 5
Result: 10 + 5 = 15
```
